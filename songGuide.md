Suno AI Song Generation Framework
This reference document defines guidelines and formatting standards to generate professional-quality song lyrics and descriptions optimized for Suno AI's music generation. It integrates structure, expressive techniques, instrument notation, and sound cues compatible with Suno AI's lyric and song prompt understanding.

1. Lyric Formatting Guidelines
Lyrics: Regular text lines - words that will be sung.

Brackets [ ]: Used for instructions or context, not sung. They describe sections, instruments, vocal types, or effects.

Parentheses ( ): Used for background vocals, echoes, whispers, ad-libs, or choral effects.

Each section (verse, chorus, bridge, etc.) starts with a bracket tag.

Example:
[Verse]
Walking through memories
[Guitar Solo]
[Chorus]
Never forget where you came from
(Yeah!)

12. LLM Prompt Guidelines
When prompting for lyric generation for Suno AI:

    *Use clear section markers ([Verse], [Chorus], etc.)
    *Include optional instructions that enhance realism ([Ad-libs:], [BPM:], [Instrumental:])
    *Avoid lengthy instructions inside the lyric body
    *Use natural, singable phrases
    *Provide accompanying short description per song

14) Input Types
a)Type:
Core: Song with vocals, length 3-5 minutes.
Long: Instrumental music, length 10+ minutes (designed to loop for extended listening, e.g., 1 hour).

b)God (Central Theme Reference):
Accept as input one of: Ram, Krishna, Shiva, Vishnu, Hanuman, Ganesha, Durga, Saraswati

The lyrics, descriptions, and themes must reflect the character, mythology, attributes, or legendary stories associated with the chosen deity.

c)Story (Song Theme or Episode)
A one-line description of the episode, legend, or devotional story inspiring the song (e.g., "Ram's exile," "Krishna's Raslila").

The LLM should incorporate thematic and narrative elements in both the lyrics (if Core) and the description.

d)Style
A musical or devotional style label, examples: "Classical Bhakti," "Classical Qawwali," "Rock Devotional."
The LLM should use this field:
    *To flavor the lyric/story structure and language.
    *To mention the style explicitly in the description.

e)Instruments
List of instruments to be featured, examples:
"Flute, Tabla, Tanpura, Soft strings"
"Sitar, Tabla, Sarangi"

The music description must mention each instrument and describe the sonic feel and arrangement accordingly.

f) Language
Hindi or English for the generation of following:
Song Title, Youtube Title, Youtube description and Youtube tags
